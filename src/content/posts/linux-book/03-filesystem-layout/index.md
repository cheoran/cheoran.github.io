## 한줄 요약

리눅스의 “폴더 지도”를 익혀서 필요한 파일을 빠르게 찾는다.

## 핵심 경로

**/etc**
설정 파일이 모여 있는 곳이다. 서비스가 왜 안 뜨는지 확인할 때 가장 먼저 보는 폴더다.

**/var/log**
로그가 저장되는 곳이다. 에러 원인을 찾을 때 여기를 확인한다.

**/var/lib**
서비스가 상태/데이터를 저장하는 곳이다. 예: DB 데이터, 캐시, 서비스 메타정보.

**/usr/bin**
일반 사용자가 실행하는 명령어가 들어 있다.

**/bin, /sbin**
시스템이 꼭 필요한 기본 명령(관리자 도구 포함)이 들어 있다.

**/home**
사용자 홈 디렉터리. 개인 파일과 설정이 저장된다.

왜 중요한가?
리눅스는 “폴더 역할이 정해져 있는 운영체제”라서, 어디를 봐야 하는지 알면 문제 해결이 빨라진다.

---

## 필수 명령어

- `ls -al`, `pwd`, `cd` : 위치/목록 확인
- `df -h`, `du -sh`, `lsblk` : 디스크 사용량 확인
- `mount`, `umount` : 마운트/언마운트

---

## 실습 1: 디렉터리 구조 탐색

```bash
lin> ls -al /
lin> ls -al /etc
lin> ls -al /var/log
```

각 폴더가 어떤 역할인지 확인한다.

---

## 실습 2: 디스크 사용량 확인

```bash
lin> df -h
lin> du -sh /var/log
lin> lsblk
```

`df`는 디스크 전체, `du`는 폴더 크기다.

---

## 체크포인트
