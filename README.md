# cheoran.github.io

Personal blog for Cheolhwan Kim.

## Stack

- Astro
- Tailwind CSS
- GitHub Pages

## Local Development

```bash
pnpm install
pnpm dev
```

## Content

Posts live in `src/content/posts/`.

Example frontmatter:

```yaml
---
title: "Post Title"
published: 2025-03-08
description: "Short summary"
image: ./cover.jpg
tags: ["Tag1", "Tag2"]
category: "Category"
draft: false
---
```

## Deploy

Pushed to `main`, deployed via GitHub Actions to GitHub Pages.
